<h1>Installation and running:</h1>

```bash

#Installation
git clone https://github.com/michaljaz/web-minecraft
cd web-minecraft
npm i

#Coffeescript transpilation
npm run coffee

#Development server
npm start

#Start node server
npm run server

```
